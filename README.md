# influencer-app

## Project setup
```
yarn install
```

## Dev Stylesheets

```bash
cd ~/src/assets/css/
compass watch --poll
```

```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
