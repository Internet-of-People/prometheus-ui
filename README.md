# influencer-app

## Project setup
```
yarn install
```

## Dev Stylesheets

```bash
cd src/assets/css/
compass watch --poll
```

### Compile and hot-reload for development
```
yarn run serve
```

### Compile and minify for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lint and fix files
```
yarn run lint
```
